# Plataforma de gestión de citas y servicios para profesionales autónomos y PYMES

Repositorio público de **documentación** del Proyecto Integrado del ciclo de Desarrollo de Aplicaciones Web (DAW).

Aquí se irán subiendo las diferentes entregas del módulo de Proyecto, así como esquemas, diagramas y cualquier documentación relevante.  
El código fuente de la aplicación se mantendrá en un repositorio privado para proteger la propiedad intelectual del proyecto.

## Estructura prevista

- `/entregas`  
  Carpeta con las entregas oficiales del módulo (E0, E1, E2, E3) en PDF u otros formatos requeridos.

- `/bocetos`  
  Wireframes, diagramas de flujo, esquemas de arquitectura, etc.

- `/notas`  
  Notas adicionales, resúmenes y decisiones de diseño tomadas durante el desarrollo del proyecto.

## Entregas

- `PI_E0` – Elección y descripción inicial del proyecto (ya entregada al centro).

El contenido y la estructura podrán ampliarse y ajustarse a medida que avance el proyecto.
